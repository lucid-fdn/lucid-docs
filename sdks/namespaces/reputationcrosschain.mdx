# Reputationcrosschain SDK

Cross-chain reputation operations.

## Operations

| Method | Description |
| --- | --- |
| `lucidGetUnifiedReputation` | Unified cross-chain reputation score |

| `lucidGetReceiptBasedReputation` | Sybil-resistant receipt-based reputation |
