### lucidGetAgentRevenue

Retrieve the accumulated revenue pool for an agent, including total earned, pending distribution, and the share token configuration if launched.


#### Example Usage


```typescript

const raijinLabsLucidAi = new RaijinLabsLucidAi();

async function run() {
  const result = await raijinLabsLucidAi.agents.revenue.lucidGetAgentRevenue({
    agentId: "<id>",
  });

  console.log(result);
}

run();
```

proportionally based on their holdings.

  const result = await raijinLabsLucidAi.agents.revenue.lucidTriggerAgentAirdrop({
    agentId: "<id>",
  });
