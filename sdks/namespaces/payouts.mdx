### calculate

Calculate a revenue split for a set of recipients using basis-point math. Default split is 70% compute, 20% model, 10% protocol (configurable via request body).


#### Example Usage


```typescript
import { RaijinLabsLucidAi } from "raijin-labs-lucid-ai";

const raijinLabsLucidAi = new RaijinLabsLucidAi();

async function run() {
  const result = await raijinLabsLucidAi.payouts.calculate({
    runId: "<id>",
    totalAmountLamports: 721074,
    computeWallet: "<value>",
  });

  console.log(result);
}

run();
```

### createFromReceipt

Extract token usage data from a receipt and compute the payout split automatically. Uses the receipt's model and compute passport IDs to resolve the split configuration.


#### Example Usage


```typescript
import { RaijinLabsLucidAi } from "raijin-labs-lucid-ai";

const raijinLabsLucidAi = new RaijinLabsLucidAi();

async function run() {
  const result = await raijinLabsLucidAi.payouts.createFromReceipt({
    runId: "<id>",
    tokensIn: 844317,
    tokensOut: 600261,
    pricePer1kTokensLamports: "<value>",
    computeWallet: "<value>",
  });

  console.log(result);
}

run();
```
